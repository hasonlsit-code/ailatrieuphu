# 💎 Ai Là Triệu Phú — Triết học Mác-Lênin (MLN121)

Trò chơi **Ai Là Triệu Phú** phiên bản đặc biệt về **Triết học Mác - Lênin**, xây dựng bằng HTML/CSS/JavaScript thuần — không cần cài đặt gì, mở là chơi được ngay.

## 🎮 Chơi ngay

👉 **https://hasonlsit-code.github.io/ailatrieuphu/**

## ✨ Tính năng

- 🎬 **Màn giới thiệu chuyên nghiệp**: Triết học là gì? → Triết học Mác - Lênin → Luật chơi
- 🪜 **15 câu hỏi** độ khó tăng dần, thang tiền thưởng từ 200.000đ đến 150.000.000đ
- 📚 **Ngân hàng 60 câu hỏi** MLN121 (dễ / trung bình / khó), mỗi ván xáo trộn ngẫu nhiên cả câu hỏi lẫn thứ tự đáp án — xem toàn bộ câu hỏi + đáp án tại [CAUHOI.md](CAUHOI.md)
- 🛟 **3 quyền trợ giúp**: 50:50 · 📞 Gọi điện cho người thân · 👥 Hỏi ý kiến khán giả
- ⏱️ Đồng hồ **30 giây** mỗi câu, hiệu ứng âm thanh, mốc an toàn tại câu 5 và câu 10
- 💡 **Giải thích đáp án sau mỗi câu** — vừa chơi vừa ôn thi
- 📱 Giao diện responsive, chơi tốt trên cả điện thoại

## 🚀 Chạy trên máy

Chỉ cần mở file `index.html` bằng trình duyệt, hoặc:

```bash
python -m http.server 8000
# rồi mở http://localhost:8000
```

Câu hỏi bám sát giáo trình Triết học Mác - Lênin (MLN121): vấn đề cơ bản của triết học, vật chất — ý thức, hai nguyên lý và ba quy luật của phép biện chứng duy vật, lý luận nhận thức — thực tiễn, và chủ nghĩa duy vật lịch sử.
